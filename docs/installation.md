# Installation

### Prerequisites

- Node.js 18 or higher
- npm 6 or higher
- Docker 20 or higher
- Docker Compose 1.29 or higher

### Commands

1. Clone the repo
   ```sh
   git clone <repo-name>
   ```
2. Install NPM packages
   ```sh
    npm install
   ```
3. Duplicate dev-start-template.sh and rename it to dev-start.sh
   ```sh
    cp dev-start-template.sh dev-start.sh
   ```
4. Run dev-start.sh
   ```sh
    ./dev-start.sh
   ```
